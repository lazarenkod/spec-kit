## Pre-Launch (T-30 to T-14)

### Strategy & Positioning

```yaml
strategy:
  positioning:
    - [ ] Target audience defined
    - [ ] Value proposition finalized
    - [ ] Key differentiators identified
    - [ ] Competitive positioning documented

  messaging:
    - [ ] Core message/tagline (< 10 words)
    - [ ] Elevator pitch (30 seconds)
    - [ ] Full product description (250 words)
    - [ ] Feature benefit statements (3-5)

  channels:
    - [ ] Primary channels identified
    - [ ] Channel-specific strategies defined
    - [ ] Budget allocated per channel
    - [ ] Success metrics per channel defined
```

### Audience Prep

```yaml
audience_prep:
  email_list:
    - [ ] Waitlist segmented
    - [ ] Email sequences drafted
    - [ ] Welcome email ready
    - [ ] Launch announcement ready

  community:
    - [ ] Beta users notified
    - [ ] Ambassador/advocate program ready
    - [ ] Community posts drafted
    - [ ] Engagement strategy defined

  press:
    - [ ] Press list compiled
    - [ ] Tier 1 publications identified
    - [ ] Journalist relationships warmed
    - [ ] Embargo strategy decided
```

## Launch Prep (T-14 to T-7)

### Content & Assets

```yaml
content_assets:
  website:
    - [ ] Landing page live
    - [ ] Pricing page updated
    - [ ] Feature pages complete
    - [ ] About/team page current
    - [ ] FAQ section complete

  visual_assets:
    - [ ] Product screenshots (annotated)
    - [ ] Demo video (30-90 sec)
    - [ ] Explainer video (optional)
    - [ ] GIF demos
    - [ ] OG images (all sizes)

  written_content:
    - [ ] Blog launch post
    - [ ] Case studies (if available)
    - [ ] Help documentation
    - [ ] Comparison pages

  press_kit:
    - [ ] Press release
    - [ ] Fact sheet
    - [ ] Founder bios
    - [ ] Logo pack
    - [ ] Screenshots pack
```

### Channel Setup

```yaml
channel_setup:
  product_hunt:
    - [ ] Ship page created
    - [ ] All assets uploaded
    - [ ] Tagline written (60 chars)
    - [ ] Description written (260 chars)
    - [ ] First comment drafted
    - [ ] Gallery images (5+)
    - [ ] Hunter confirmed (if using)
    - [ ] Launch time scheduled

  social_media:
    - [ ] Twitter thread drafted
    - [ ] LinkedIn post drafted
    - [ ] Posts scheduled
    - [ ] Hashtag strategy set
    - [ ] Engagement list ready

  email:
    - [ ] Launch email finalized
    - [ ] Segment lists ready
    - [ ] Send time scheduled
    - [ ] A/B tests set up

  communities:
    - [ ] Reddit posts drafted
    - [ ] HN post drafted
    - [ ] Slack/Discord communities identified
    - [ ] Community guidelines reviewed
```

### Technical Marketing

```yaml
technical_marketing:
  analytics:
    - [ ] GA/PostHog configured
    - [ ] Conversion events defined
    - [ ] UTM scheme documented

    - [ ] CTAs optimized
```

## Launch Day (T-0)

### Launch Execution

```yaml
launch_execution:
  timing:
    "00:00 PT": "Product Hunt goes live"
    "06:00 ET": "Email to East Coast segments"
    "09:00 ET": "Twitter thread posted"
    "09:30 ET": "LinkedIn announcement"
    "10:00 ET": "Hacker News Show HN"
    "11:00 ET": "Press release distributed"
    "12:00 ET": "Email to remaining segments"
    "15:00 ET": "Community posts"
    "18:00 ET": "Evening social push"

  channels:
    - [ ] Product Hunt launched
    - [ ] Launch email sent
    - [ ] Social posts published
    - [ ] Press release distributed
    - [ ] Community posts published
    - [ ] Paid campaigns activated (if applicable)
```

### Active Engagement

```yaml
active_engagement:
  product_hunt:
    - [ ] Respond to every comment
    - [ ] Thank upvoters
    - [ ] Share progress updates
    - [ ] Answer questions promptly

  social_media:
    - [ ] Monitor mentions
    - [ ] Reply to comments
    - [ ] Retweet/share supporters
    - [ ] Post updates

  support:
    - [ ] Monitor support channels
    - [ ] Quick response to issues
    - [ ] Collect feedback
    - [ ] Document common questions
```

### Monitoring

```yaml
monitoring:
  metrics:
    - [ ] Real-time traffic
    - [ ] Signup/conversion rate
    - [ ] Product Hunt ranking
    - [ ] Social engagement
    - [ ] Email open/click rates

  alerts:
    - [ ] Site uptime
    - [ ] Error rates
    - [ ] Conversion drops
    - [ ] Negative mentions
```

## Channel-Specific Checklists

### Product Hunt

```yaml
product_hunt:
  pre_launch:
    - [ ] Optimal day selected (Tue-Thu)
    - [ ] 12:01 AM PT launch scheduled
    - [ ] Supporters notified
    - [ ] First comment ready

  day_of:
    - [ ] Monitor every 30 minutes
    - [ ] Respond to all comments within 1 hour
    - [ ] Share milestones on social
    - [ ] Thank supporters publicly

  assets_required:
    - Tagline (60 chars max)
    - Description (260 chars max)
    - Gallery images (1270x760, min 3, max 8)
    - Logo (240x240)
    - Thumbnail (1270x760)
    - Maker comment (personal story)
```

```yaml
hacker_news:
```
